# Variável do tipo array

- Nomes de variáveis devem começar com `$`.
- Nomes de variáveis não podem começar com números.
- Nomes de variáveis não podem conter espaços.

```php
<?php

$usuario = [
    'id' => 1,
    'nome' => 'Nome do Usuário',
    'email' => 'email@dominio.com',
];

?>
```

ou

```php
<?php

$usuario = array(
    'id' => 1,
    'nome' => 'Nome do Usuário',
    'email' => 'email@dominio.com',
);

?>
```
