# Estruturas condicionais

### if ... else

```php
<?php

$numero = 5;

if ($numero == 5) {
    echo 'O número é 5';
} else {
    echo 'O número é diferente de 5';
}

?>
```

### switch ... case

```php
<?php

$numero = 3;

switch ($numero) {
    case 1:
        echo 'O número é 1';
        break;
    case 2:
        echo 'O número é 2';
        break;
    default:
        echo 'O número é 3';
}

?>
```
