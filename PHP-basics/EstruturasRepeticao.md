# Estruturas de repetição

### while

```php
<?php

$contator = 0;

while ($contador < 10) {
    echo $contador, '<br>';
    $contador++;
}

?>
```

### do ... while

```php
<?php

$contator = 0;

do {
    echo $contador, '<br>';
    $contador++;
} while ($contador < 10);

?>
```

### for

```php
<?php

for ($contador = 0; $contador < 10; $contador++) {
    echo $contador, '<br>';
}

?>
```
